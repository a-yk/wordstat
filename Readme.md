# WORDSTAT #

Приложение рассчитывает как часто буквы появляются в тексте.

### Как запустить ###

* cклонировать репозиторий;
* переименовать файл .env.dist в .env
* запустить команду `cd wordstat && composer update && php -S localhost:8000 -t ./public/`
